# laissa_naildesigner

Página de links estilo Linktree, feita do zero com HTML e CSS puro.

Funcionalidades

Links para redes sociais com ícones oficiais
Botão de WhatsApp com mensagem pré-definida
Animação suave ao carregar a página

🗂️ Estrutura do projeto
projeto/

├── README.md         # Este arquivo
└── imagens/          # Pasta com as imagens
    ├── avatar.png    # Foto de perfil
    └── favicon.png   # Ícone da aba do navegador
